Artifact¶

Artifact can be entered here, e.g. a coin

On the index page already entered entities are listed in a Table.

  • Click on the + button to enter a new one.
  • Click on the name of an entry in the list to access the detail view.
  • To edit or delete an entry click on the Edit button in the detail view.

Form fields¶

  • Name
  • Type
  • Date
  • Description
  • Map
  • Reference System

Can be linked via tabs to¶

  • Source - when it is referenced there
  • Event - when it was moved at a Move event
  • Reference
  • File

If the artifact is a carrier of information it can be linked to a source. For this you would have to edit or create the source and choose the artifact within the source form.
