OpenAtlas manual
================

OpenAtlas is an open source application for archeological, historical and geospatial data.

This is the manual for the user interface. You'll find more information about the project on the
OpenAtlas `website <https://openatlas.eu>`_ and development specific documentation in our
Redmine `Wiki <https://redmine.openatlas.eu/projects/uni/wiki/Wiki>`_.
